# JAVA로 만든 단어 입력 게임
---
 프로젝트 소개<br>
 - 2학년 객체지향언어2 기말고사 프로젝트 입니다. 프로젝트의 주제는 단어 입력게임(산성비게임) 입니다.<br><br>
 - 프로젝트의 목적은 멀티스레딩의 활용과 Swing GUI, 텍스트 입출력, 각종 프로그래밍 능력을 활용한 산성비 게임의 구현입니다.
 2학년의 수준이라 아직 코드의 구조와 깔끔하지 못하고 중복된 부분도 많이 있습니다. 
---
> 프로젝트 개요 <br><br>
 게임의 배경은 우주공간입니다. 운석을 파괴하면서 지구를 지키는 컨셉이며, 플
레이어는 주어진 시간 내에 하트를 잃지 않으면서 최대한 많은 단어(운석)를 입
력해서 높은 점수를 얻는 것이 목표입니다. <br><br>
> 난이도는 4가지(초보, 중수, 고수, 개발자)가 있으며 각각 난이도별로 단어가 내려오는 주기가 점점 빨라집니다. 단어
가 내려오는 속도는 처음엔 모두 동일 합니다. 하지만 점수가 높아질수록 단어의 
낙하 속도가 증가합니다. 정해진 확률대로 단어 운석이 아닌 UFO와 보급 상자가 
내려옵니다. UFO의 단어를 입력하면 단어의 낙하 속도가 일정량 감소하게 됩니
다. 보급 상자의 단어를 입력하면 추가시간 5초와 만약 하트를 잃었을 경우, 하
트를 한 개 채워줍니다. 추가로 단어가 생성되고 빨리 입력한 시간과 비례해서 
점수가 오릅니다.  <br><br>
> 게임이 끝나면 플레이어의 점수와 랭크가 표시됩니다. 점수판에
는 5명의 최고 기록이 저장되어 있는데 만약 기록을 경신할 경우 자동으로 새로
고침이 됩니다.
